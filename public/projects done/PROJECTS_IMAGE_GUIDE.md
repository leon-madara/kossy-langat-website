# Project Image Guide — EPS 3D Structural Engineering Website

> **Purpose:** This document guides Claude Code in understanding all project images, their descriptive names, and how to use them correctly within a structural engineering portfolio website that showcases EPS 3D panel construction technology.

---

## About the Firm & Technology

This is a **structural engineering firm** specialising in **EPS 3D construction systems** — a modern, efficient building method that uses Expanded Polystyrene (EPS) panels reinforced with steel wire mesh on both faces, filled with sprayed concrete (shotcrete). EPS 3D delivers lighter structures, faster build times, better thermal performance, and cost savings compared to traditional masonry or concrete slab construction.

The portfolio spans **commercial, residential, and multi-unit** project types across Nairobi and Kajiado County, Kenya.
